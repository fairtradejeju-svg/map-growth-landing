Save and Exit 같은 그룹 재시험 시
세션을 저장합니다. 나중에 같은 학생 그룹으로 다시 시험할 수 있습니다.
Delete Session 다른 그룹 구성 시
세션을 삭제합니다. 시험 결과에는 영향 없습니다. 새 그룹을 구성할 때 선택하세요.
선택:
  • Save and Exit — 같은 그룹으로 나중에 다시 시험할 경우
  • Delete Session — 다른 그룹을 구성할 경우 (시험 결과에는 영향 없음)
Save and Exit vs Delete Session 차이
둘 다 시험 결과에는 영향이 없습니다. 차이는 세션 자체를 보관하느냐 삭제하느냐입니다.
같은 학생 그룹으로 다른 과목 시험을 볼 예정이면 Save and Exit이 편합니다.

부록

한 학생의 시험만 교체하려면

  • 해당 학생 선택 → Test Again

  • 학생에게 같은 세션 이름과 비밀번호로 다시 로그인 안내
