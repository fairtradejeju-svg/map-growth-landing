화면 구성

화면 번호별 설명

1
사이드바 메뉴
왼쪽 메뉴에서 VIEW REPORTS, MANAGE STUDENTS 등 다른 기능으로 이동할 수 있습니다.
2
세션 정보 바 -- Session Name, Password, Proctor PIN
Session Name과 Password를 학생에게 안내합니다. 칠판이나 화면에 적어주세요.
Proctor PIN은 Rapid Guessing 발생 시 학생 기기에 입력해야 하는 번호입니다. 미리 메모해두세요.
View PDF를 클릭하면 Testing Tips 문서를 볼 수 있습니다.
3
Session Details + Action Needed + Rapid-Guessing Alerts
Session Details -- 전체 학생 수, 현재 시험 중인 학생 수
Action Needed -- Confirm 대기 중, Paused 학생 수. Confirm 버튼으로 대기 학생 확인
Refresh Page -- 60초마다 자동 갱신. 수동으로 눌러 즉시 갱신 가능
Rapid-Guessing Alerts -- 빠른 추측으로 일시정지된 학생 수 표시
4
도구 바 -- 학생 관리 버튼들
Select Status -- 상태별 필터링 (정렬 용도)
Select Action -- 학생 체크박스를 먼저 클릭해야 활성화! Pause, Resume, Suspend, Test Again, Terminate 등 실행
Assign Test -- 시험 배정
Assign Accommodations -- 편의사항 설정 (시간 연장 등)
Add/Remove Student -- 학생 추가/제거
5
학생 테이블
학생 이름, Status(상태), Question #(진행률), Test Assignment(배정 시험), Accommodations, Rapid Guesses 표시.
학생 이름 앞의 체크박스를 클릭하면 위 도구 바의 Select Action이 활성화됩니다.
7
END SESSION 버튼
시험 전체를 종료합니다. 자세한 내용은 04-2 시험종료 가이드를 참고하세요.
상태 정보

학생 상태 (Status) 의미

상태의미감독관 할 일
Testing 시험 진행 중 정상. 지켜보기만 하면 됩니다.
Completed 시험 완료 정상 완료. 추가 시험 배정 가능 (Test Again → Assign Test)
Paused 일시정지 (최대 25분) 25분 내 Resume. 초과 시 자동 Suspend.
Rapid Guessing 빠른 추측 경고 -- 자동 일시정지됨 학생에게 가서 신중히 답하도록 안내 → Proctor PIN 입력하여 재개
Suspended 중단됨 (최대 28일 내 재개 가능) 당일: Test Again / 다른 날: 새 세션에서 Resume Test
동작 설명

Select Action 동작 설명

학생을 선택한 후 Select Action ▾에서 선택할 수 있는 동작입니다.

동작기능사용 시기
Pause 시험 일시정지 (최대 25분) 화장실, 잠깐 쉬는 시간
Resume Pause 상태 학생 시험 재개 Pause 후 돌아왔을 때
Suspend 시험 저장 후 중단 (28일 내 재개) 학생이 아프거나, 다른 날 이어서 볼 때
Test Again Suspend된 학생 당일 재개 Suspend 후 같은 날 다시 볼 때
Terminate 시험 완전 삭제 -- 모든 답안 삭제! 잘못된 시험이 배정되었을 때만
Terminate는 모든 답안을 삭제합니다!
잘못된 시험 배정 시에만 사용하세요. 되돌릴 수 없습니다.
학생 PC가 튕겼을 때 (인터넷 끊김, 브라우저 종료 등)
1
Proctor 콘솔에서 해당 학생 선택
학생 이름 또는 체크박스를 클릭합니다.
2
Select Action → Suspend
먼저 시험을 안전하게 저장합니다. 답안이 보존됩니다.
3
Select Action → Test Again
Suspend 후 바로 Test Again을 눌러 시험을 재개합니다.
4
학생이 다시 로그인 → 중단된 곳부터 이어서 진행
같은 Session Name과 Password로 다시 접속하면 됩니다.
핵심 기억 포인트
트러블슈팅 05번 가이드에도 나오지만, 이 상황은 자주 발생하므로 여기서 다시 안내합니다.
Suspend → Test Again 순서만 기억하세요. 답안은 안전하게 보존됩니다!
Rapid Guessing 발생 시 대처
학생이 문제를 읽지 않고 빠르게 답을 클릭하면 자동으로 시험이 멈춥니다.
1
콘솔에서 Proctor PIN 확인
화면 2번 영역에 표시된 PIN 번호를 확인합니다.
2
학생에게 가서 안내
"천천히 문제를 읽고 답해주세요" 라고 격려합니다.
3
학생 기기에 PIN 입력 → Resume Test
학생 화면에 PIN 입력란이 뜹니다. PIN을 입력하고 Resume Test를 누르면 시험이 재개됩니다.
3회 반복 시 주의
3회 반복 시 해당 학생의 시험을 Suspend하고 다른 날 다시 진행하는 것을 권장합니다.
