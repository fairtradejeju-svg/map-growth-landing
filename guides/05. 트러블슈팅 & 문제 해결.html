05 — 부록

트러블슈팅 & 문제 해결

시험 중 자주 발생하는 문제와 해결 방법을 상황별로 정리했습니다. 당황하지 마시고, 아래 가이드를 따라 해결해 보세요.

5개 카테고리 상황별 즉시 대응 단계별 해결법
로그인 문제
Q
학생이 로그인했는데 Proctor 콘솔 목록에 안 보여요

학생의 현재 상태에 따라 조치가 다릅니다:

1
Confirmed 상태 → Select Action → Do Not Confirm → 다시 Confirm
2
Testing 상태 → Select Action → Suspend → Test Again
3
Suspended 상태 → Select Action → Test Again → 학생에게 다시 로그인 안내 → Confirm
Q
"Students are Ineligible to Test" 메시지가 나와요

시험이 "학기당 1회"(Once Per Term)로 제한되어 있을 수 있습니다.

→ District Assessment Coordinator에게 시험 제한 설정 확인을 요청하세요.

시험 배정 문제
Q
잘못된 시험이 배정되었어요

아직 시험 시작 전 (To Be Confirmed):

→
Select Action → Do Not Confirm → Assign Test로 올바른 시험 재배정

이미 시험 중 (Testing):

→
Select Action → Suspend (나중에 볼 수도 있으면) 또는 Terminate (완전 취소)
→
Test Again → 올바른 시험 재배정
Terminate 주의
Terminate는 모든 답안을 삭제합니다. 신중하게 사용하세요.
화면 / 기술 문제
Q
학생 PC가 튕겼어요 (인터넷 끊김, 브라우저 종료)
1
Proctor 콘솔에서 해당 학생 이름 앞의 체크박스 클릭하여 선택
2
상단 두 번째 칸 Select Action이 활성화됨 → 클릭 → Suspend 선택 (답안 안전 저장)
3
Select Action → Test Again
4
학생이 같은 Session Name/Password로 다시 로그인 → 중단된 곳부터 이어서 진행
안심하세요
답안은 안전하게 보존됩니다. Suspend → Test Again 순서만 기억하세요.
Q
문제 화면이 하얗게 나와요 (Blank Screen)
1
Proctor 콘솔에서 해당 학생 → Select Action → Suspend
2
학생 기기에서 테스트 브라우저 완전 종료
3
Proctor 콘솔에서 → Select Action → Test Again
4
학생이 다시 로그인 → 중단된 곳부터 이어서 진행
Q
시험이 너무 느려요

학생 기기에서 화면을 새로고침하세요:

PC
F5 또는 Reset 버튼 클릭
Mac
Cmd + R
Q
결함이 있는 문제를 건너뛰고 싶어요
1
해당 학생 → Select Action → Pause
2
다시 해당 학생 → Select Action → Resume
3
학생이 Resume을 누르면 새 문제가 표시됩니다
문제 신고 방법
문제 자체를 신고하려면: 학생 기기에서 Ctrl + Shift + P → Proctor PIN 입력 → 문제 상황 설명
중단된 시험 이어서 보기
Q
Suspend된 시험을 이어서 보려면?

같은 날, 같은 세션:

→
해당 학생 → Select Action → Test Again

다른 날:

1
저장된 세션을 다시 열거나 새 세션 생성
2
Find Students to Test → Test History Search 탭
3
학교, 시험 등 필터 → Search → 학생 추가
4
시험 배정 → Test Now → Confirm 시 Resume Test 선택
재개 기한
14일 이내에 재개하는 것을 권장합니다 (최대 28일 가능).
