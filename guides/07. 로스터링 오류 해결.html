CSV 형식이 잘못됨 (UTF-8로 저장)Excel에서 CSV(쉼표 구분)으로 재저장
컬럼/헤더가 템플릿과 다름NWEA 템플릿의 헤더 행을 복사하여 붙여넣기

한국어 Windows에서 구분자가 세미콜론제어판에서 목록 구분 기호를 쉼표(,)로 변경

유형 2
File Format 오류 — 데이터 내용 문제
오류 메시지원인해결
"XXX cannot be the same for more than one YYY" 서로 다른 사람이 같은 ID/사용자명 공유 Column Name, Row Number 확인 후 중복 제거
"Field cannot be empty" 필수 컬럼에 빈 값 해당 행과 열을 찾아 빠진 정보 입력
"Date of Birth must be a valid date" 날짜 형식 오류 MM/DD/YYYY 형식으로 통일
팁
오류가 많을 때는 해당 열을 필터링하여 문제 데이터만 골라 수정하면 빠릅니다.

Reconcile 오류 — 기존 데이터와 충돌

이전에 등록된 데이터와 현재 업로드 데이터가 일치하지 않을 때 발생합니다.
