Chrome 실시간 번역 자막 설정 (1회만)
  1. Chrome 브라우저에서 영상 링크 클릭
  2. 주소 표시줄 오른쪽 ⋮ (더보기) → 설정 → 접근성
  3. "실시간 자막" 켜기 → "번역" 토글도 켜기
  4. 번역 언어를 한국어로 설정
  5. 영상 재생 시 하단에 한국어 자막이 실시간 표시됩니다
