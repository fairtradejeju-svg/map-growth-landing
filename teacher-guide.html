자주 묻는 질문

테스트 운영 중 문제가 생겼나요?

자주 발생하는 문제와 해결 방법을 정리했습니다.

경고창이 뜨면서 테스트가 시작되지 않아요

팝업 차단이 활성화되어 있을 가능성이 높습니다.

  1. 크롬 브라우저 주소창 오른쪽의 팝업 차단 아이콘을 클릭
  2. "이 사이트의 팝업 항상 허용" 선택
  3. 페이지를 새로고침(F5) 후 다시 시도
"Some student(s) are unable to test" 경고

학생 계정이 테스트 세션에 제대로 등록되지 않았을 때 나타납니다.

  1. MAP 관리자 페이지에서 해당 학생의 세션 배정 확인
  2. 학생 이름과 ID가 정확한지 확인
  3. 세션을 재시작하거나 학생을 다시 추가
  4. 문제가 지속되면 cs@mapgrowthsupportkorea.com으로 문의
테스트 중 에러 메시지가 나타났어요

"We encountered a temporary error" 메시지는 일시적인 서버 문제입니다.

  1. 당황하지 마세요 — 학생의 진행 상황은 자동 저장됩니다
  2. 5~10초 기다린 후 "다시 시도" 버튼 클릭
  3. 반복되면 인터넷 연결 상태를 확인
  4. 그래도 안 되면 브라우저를 닫고 다시 로그인하면 이어서 진행됩니다
해상도 오류 (1024x768)가 나와요

MAP 테스트는 최소 1024x768 해상도가 필요합니다.

  1. 크롬에서 Ctrl+0 (또는 Cmd+0)으로 확대/축소를 100%로 리셋
  2. 전체 화면 모드(F11)로 전환
  3. 태블릿의 경우 가로 모드로 회전
  4. 화면이 작은 기기에서는 외부 모니터 연결을 권장합니다
비밀번호를 잊어버렸어요

교사 계정 비밀번호 재설정 방법입니다.

  1. MAP 로그인 페이지에서 "Forgot Password" 클릭
  2. 등록된 이메일 주소 입력
  3. 메일로 받은 링크를 클릭하여 새 비밀번호 설정
  4. 메일이 안 오면 스팸 폴더를 확인하거나 저희에게 문의해 주세요
어떤 브라우저를 사용해야 하나요?

크롬(Chrome) 브라우저를 사용해 주세요.

  1. MAP Growth는 Chrome에 최적화되어 있습니다
  2. Safari, Firefox, Edge에서는 일부 기능이 제한될 수 있습니다
  3. Chrome이 설치되어 있지 않으면 google.com/chrome에서 다운로드
  4. 항상 최신 버전으로 업데이트해 주세요
"Students are Ineligible to Test" 메시지가 표시될 경우

프로그램 재 설정(학기 변경)이 필요합니다. CS팀으로 연락주시길 바랍니다.

  1. cs@mapgrowthsupportkorea.com 으로 문의
  2. 또는 카카오톡 채널로 문의
"Action Needed to Continue" 메시지가 표시될 경우

일시 중단된 시험을 이어서 진행해야 합니다.

  1. Resume Test — 일시 중단된 시험을 이어서 진행
  2. Start Test Over — 시험을 처음부터 다시 시작 (기존 기록 삭제)
  3. Do Not Confirm — 기존 시험을 종료

테스트 운영 시 알아두세요

⚡ 테스트 전 체크리스트

크롬 브라우저를 사용해 주세요. 가능하면 NWEA Secure Testing Browser 설치를 권장합니다.
관리자 ID는 초기 설정한 이메일 주소입니다. start.mapnwea.org에서 비밀번호를 설정해 주세요.
테스트 전 check.nwea.org에서 시스템 진단을 실행하세요.
아이패드는 앱스토어에서 "NWEA"를 검색하여 전용 앱을 설치하세요.
테스트 후 점수는 화면에 바로 표시됩니다. 정식 리포트는 48시간 후에 확인 가능하므로, 점수 화면을 핸드폰으로 촬영해 기록해 두세요.
테스트가 끝나면 반드시 End Session을 클릭하세요. 누르지 않으면 다음 테스트 시작 시 에러가 발생합니다.
